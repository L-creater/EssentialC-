stringstream

stringstream是一个C++底下的类别，专门拿来读取字串并且处理，很多时候拿来做字串的切割，或者是int跟string类别之间的转换。把int型态的数字转成string此时的stringstream就像个桥梁，可以负责当中间转换的部分。

stringstream 提供了 >> 与 << 运算子来读取或写入:
<< 代表写入 **stringstream** 中，>> 代表从 stringstream 拿出。


// swapping ostringstream objects
#include <iostream>  // std::cout
#include <ostream>
#include <sstream>  // std::stringstream
#include <string>   // std::string

int main() {
  std::stringstream ss;
  // 将 100 和200 输入到ss中
  ss << 100 << ' ' << 200;
  ss << "example string";
  int foo, bar;
  std::string s;
  ss >> foo >> bar >> s;

  std::cout << "foo: " << foo << '\n';
  std::cout << "bar: " << bar << '\n';
  std::cout << "s: " << s << '\n';

  // ss.str("example string");

  std::string s1 = ss.str();
  std::cout << "s1: " << s1 << '\n';
  return 0;
}

foo: 100

bar: 200

s: example

s1: 100 200example string
