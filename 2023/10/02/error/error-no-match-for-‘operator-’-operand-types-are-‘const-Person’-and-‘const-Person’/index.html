error: no match for ‘operator==’ (operand types are ‘const Person’ and ‘const Person’)
发布于：2023-10-02 15:48:37 标签：/ error / 访问：

错误报告

/usr/include/c++/11/bits/stl_pair.h:467:51: error: no match for ‘operator==’ (operand types are ‘const Person’ and ‘const Person’)
  467 |     { return __x.first == __y.first && __x.second == __y.second; }
      |                                        ~~~~~~~~~~~^~~~~~~~~~~~~
/home/lyt/my_sylar/tests/test_config.cc:129:8: note: candidate: ‘bool Person::operator==(const Person&)’ (near match)
  129 |   bool operator==(const Person &oth) {
      |        ^~~~~~~~
/home/lyt/my_sylar/tests/test_config.cc:129:8: note:   passing ‘const Person*’ as ‘this’ argument discards qualifiers

这个错误表明在某个地方尝试比较两个 const Person 对象，但是编译器找不到适当的 operator== 实现。错误信息还提到了一个备选项，它是你自己为 Person 类实现的 operator== 函数，但它可能没有被正确地声明为 const 函数。

为了解决这个问题，你需要确保你的 Person 类中的 operator== 函数被声明为 const 成员函数，因为你正在比较两个 const Person 对象。

错误代码

class Person {
 public:
  Person(){};
  std::string m_name;
  int m_age = 0;
  bool m_sex = 0;

  std::string toString() const {
    std::stringstream ss;
    ss << "[person name= " << m_name << " age= " << m_age << " sex= " << m_sex << "]";
    return ss.str();
  }
  bool operator==(const Person &oth) {
    return m_name == oth.m_name && m_sex == oth.m_sex && m_age == oth.m_age;
  }
};

修改后的代码

class Person {
 public:
  Person(){};
  std::string m_name;
  int m_age = 0;
  bool m_sex = 0;

  std::string toString() const {
    std::stringstream ss;
    ss << "[person name= " << m_name << " age= " << m_age << " sex= " << m_sex << "]";
    return ss.str();
  }
  bool operator==(const Person &oth) const {
    return m_name == oth.m_name && m_sex == oth.m_sex && m_age == oth.m_age;
  }
};
