error: invalid new-expression of abstract class type ‘sylar::IPAddress’
发布于：2023-10-25 22:27:11 标签：/ error / sylar / 访问：

error:

error: invalid new-expression of abstract class type ‘sylar::IPAddress’
  162 |         { ::new((void *)__p) _Up(std::forward<_Args>(__args)...); }
      |           ^~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
bool Address::GetInterfaceAddresses(std::vector<std::pair<Address::ptr, uint32_t>> &result,
                                    const std::string &iface, int family) {
  if (iface.empty() || iface == "*") {
    if (family == AF_INET || family == AF_UNSPEC) {
      // result.push_back(std::make_pair(Address::ptr(new IPv4Address()), 0u));
      result.push_back(
          std::make_pair(std::dynamic_pointer_cast<Address>(std::make_shared<IPAddress>()), 0u));
    }
    if (family == AF_INET6 || family == AF_UNSPEC) {
      result.push_back(std::make_pair(Address::ptr(new IPv6Address()), 0u));
    }
    return true;
  }
  ...
}
  1. 这个报错代表一个尝试在实例化一个抽象类，也就是说父类的接口中有纯虚函数在子类中没有实现；
  2. 实例化了父类，而父类中包含了纯虚函数，virtual　＝０是不能实例化的，相当于抽象类只能继承，不能实例化． —>这里IPAddress,Address都是抽象类，IPAddress是address的子类。

解决方法：　实例化子类　 —>IPv4Address

应该修改为：

bool Address::GetInterfaceAddresses(std::vector<std::pair<Address::ptr, uint32_t>> &result,
                                    const std::string &iface, int family) {
  if (iface.empty() || iface == "*") {
    if (family == AF_INET || family == AF_UNSPEC) {
      // result.push_back(std::make_pair(Address::ptr(new IPv4Address()), 0u));
      result.push_back(
          std::make_pair(std::dynamic_pointer_cast<Address>(std::make_shared<IPv4Address>()), 0u));
    }
    if (family == AF_INET6 || family == AF_UNSPEC) {
      result.push_back(std::make_pair(Address::ptr(new IPv6Address()), 0u));
    }
    return true;
  }
  ...
}
