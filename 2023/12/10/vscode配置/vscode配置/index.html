vscode 设置

Ctrl + , 进入设置

搜索brack

  1. 第一个使得括号有颜色。
  2. 第二个可以看到括号之间的内容区间。如下图所示

搜索smooth

前面的四个选项，全部打开，或感觉到屏幕滑动,和光标的移动更加的顺滑。

最先下面的选项，选择smooth，则会发现光标的闪烁变了。

搜索preview


将图中选中的选项打开，会出现预览的效果，类似zsh-autosuggestions插件

搜索code lens


关闭下面的两个选项

设置切换终端的快捷键

ctrl+shift+p 进入命令面板，搜索keyboard。选择图片中高亮的选项。

打开 keybindings.json,进行编辑。

再此文件中添加如下内容.

// 将键绑定放入此文件中以覆盖默认值
[
    {
        "key": "ctrl+tab",
        "command": "workbench.action.terminal.focusNext", // 切换到下一个终端 
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+w",
        "command": "workbench.action.terminal.kill", // 关闭当前终端
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+t",
        "command": "workbench.action.terminal.new", // 打开新的终端
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+1",
        "command": "workbench.action.terminal.focusAtIndex1", // 打开终端1
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+2",
        "command": "workbench.action.terminal.focusAtIndex2",
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+3",
        "command": "workbench.action.terminal.focusAtIndex3",
        "when": "terminalFocus"
    },
    {
        "key": "ctrl+4",
        "command": "workbench.action.terminal.focusAtIndex4",
        "when": "terminalFocus"
    }
]

Ctrl+ , 进入设置界面，搜索 Tab Size 设置为 2
